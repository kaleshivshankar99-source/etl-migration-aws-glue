# ETL Migration from IBM DataStage to AWS Glue

## 📌 Project Overview

This project demonstrates the migration of legacy IBM DataStage ETL workflows to AWS Glue using PySpark.

The ETL pipeline extracts data from CSV files, applies business transformations, validates the data using SQL, and prepares it for analytics.
